# GitCMS v1.1.43 Simple Diagnostics

UI cleanup.

## Changed

Diagnostics is now simple by default.

The visible summary shows:

```txt
Admin version
Content repo
Content branch
Main branch
Cache status
Loaded content source
Unsaved fragments
Validation warnings
```

Full debug details are still available under:

```txt
Advanced diagnostics
```

## Why

The previous Diagnostics view was useful for debugging but too noisy for normal use.

## Debug status

Diagnostics/cache work is considered complete for now. Future work should move to other product tasks unless a new bug appears.
