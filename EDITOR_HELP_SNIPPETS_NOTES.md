# GitCMS v1.1.44 Editor Help Snippets

Editor usability improvement.

## Added

A collapsible editor help panel below the raw HTML textarea:

```txt
HTML help / snippets
```

It includes snippet buttons for:

```txt
Heading
Paragraph
Intro text
Button link
Image
List
Card
Section wrapper
Two columns
Quote
```

## Behavior

Clicking a snippet inserts raw HTML at the cursor. If text is selected, snippets use the selected text where appropriate.

## No logic changes

No changes to:

```txt
save
publish
content loading
cache
GitHub API
```
