GitCMS

Zero-backend CMS. Run this admin anywhere; edit a separate site/content GitHub repo.

This is the repository containing docs/, fragments.json, and the content branch. It can be different from the admin-hosting repo.
Use a fine-grained PAT scoped to the content/site repo with Contents: Read & write. A classic repo-scoped token works but can write to every repo you own. Stored in sessionStorage for the current browser session. Older localStorage tokens are migrated once and removed.
